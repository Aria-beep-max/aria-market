距10日均線 %
現在股價比10日均線高或低幾%。你的「低於均線-5% 觸發 Alert」邏輯就是這個數字的延伸，讓你不用自己算，一眼看出還差多遠。

負值 = 股價在均線下方（跌破均線）
正值 = 股價在均線上方（強勢）
// 例：-8.2% 代表快接近你的 Alert 觸發點了
CORE
⭐⭐⭐
RSI（相對強弱指標）
0~100 的數字，衡量最近 14天 的漲跌速度和力道。

RSI < 30 = 超賣，跌太快，可能即將反彈
RSI > 70 = 超買，漲太快，可能即將回調
RSI ≈ 50 = 正常狀態
// 觸發加碼時 RSI 也 < 30，信號強度大幅提升
CORE
⭐⭐⭐
MACD 訊號
兩條移動均線的差值，看趨勢轉折點。

黃金交叉：快線從下往上穿越慢線 → 可能是買點
死亡交叉：快線從上往下穿越慢線 → 可能是賣點
// 觸發加碼 + MACD 黃金交叉同時出現，是最強的買進信號
⭐⭐ 進場 & 確認指標 — 幫你判斷時機
ENTRY
⭐⭐
今日高 / 低
今天的最高價和最低價，顯示當天的波動範圍。

現價靠近今日低點 → 還在跌，再等一下
現價靠近今日高點 → 今天有反彈力道
// 用來判斷當天進場時機，不要追高
SIGNAL
⭐⭐
成交量 vs 均量 / 量比
量比 = 今日成交量 ÷ 35日均量

漲跌要有量支撐才有意義。跌破支撐但量很小可能是假訊號，不要急著進場。

量比 > 2 = 今天交易量是平常兩倍，值得注意
// 量比 > 2 且股價反彈，代表大資金在買
SIGNAL
⭐⭐
布林通道（Bollinger Bands）
以 20 日均線為中心，上下各畫一條「正常波動範圍」的線。

股價碰到下軌 → 跌過頭，可能反彈
股價碰到上軌 → 漲過頭，可能回調
通道變窄 → 大波動即將來臨
// 你的均線-5% 策略跟布林下軌概念類似，可互相印證

⭐
52週高 / 低
過去一年的最高價和最低價，讓你知道現在在歷史位置上是貴還是便宜。

現價靠近 52 週低 → 相對便宜，但確認不是繼續跌
現價靠近 52 週高 → 創新高，動能強，但也可能回調
// 搭配 RSI 一起看，判斷是真便宜還是繼續跌
REF
⭐
Beta 值
這支股票跟大盤比，波動有多誇張。

Beta = 1 → 跟大盤一樣漲跌
Beta = 1.5 → 大盤跌 10%，它跌 15%
Beta = 0.5 → 波動只有大盤一半
// QLD、SSO 是 2x ETF，Beta 很高，要特別控制部位大小
REF
⭐
均線多頭排列
5日 > 10日 > 20日 > 60日均線，全部從上到下排列整齊，代表短中長期都在上漲。

反過來叫空頭排列，代表整體趨勢向下，要謹慎進場。
// 多頭排列時進場，勝率通常比較高
REF
⭐
本益比（P/E）
花多少錢買公司每賺 1 元的獲利。P/E = 20 代表花 20 元買 1 元獲利。

P/E 低 = 相對便宜（但可能公司有問題）
P/E 高 = 市場期待高（科技股通常高）

⚠ ETF 不適用此指標
// 同產業比較才有意義，跨產業比沒太大參考價值
REF
⭐
市值
這間公司總共值多少錢，用來評估風險等級。

大型股（T 級）：台積電、蘋果，穩定，不容易暴漲暴跌
中小型股（B 級以下）：UUUU、USAR，風險高但漲起來也快
// 1T = 1兆，1B = 10億，1M = 百萬

外資連續買超 → 大資金進場，是好兆頭

⚠ 台股專用，美股無此公開資料

融資比率
散戶借錢買股（融資）的比例。

融資比例高 → 股價下跌時會有強制賣出壓力，跌更快
融資比例低 → 籌碼比較乾淨，不容易被踩踏
// 台股限定，高融資股在大跌時特別危險
REF
△
殖利率（Dividend Yield）
每年配多少股息相對於股價的比例。

高殖利率（如 AEM）→ 適合長期存股
成長股（NVDA、PLTR）→ 通常殖利率很低或沒有

⚠ 你的策略主要是價差，殖利率參考就好
